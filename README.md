# 🎙️ Assistente de Voz com Python

Projeto desenvolvido em parceria com a DIO (Digital Innovation One), com foco em aprendizado prático de Inteligência Artificial e automação com Python.

## 🚀 Funcionalidades

* 🎤 Captura de áudio do usuário
* 🧠 Processamento com IA
* 💬 Geração de respostas inteligentes
* 🔊 Reprodução de áudio

## 🛠️ Tecnologias utilizadas

* Python
* Google Colab
* OpenAI API
* Bibliotecas de áudio

## 📚 Objetivo

Este projeto foi criado com o objetivo de aplicar conceitos de:

* Inteligência Artificial
* Automação
* Processamento de linguagem natural

## ▶️ Como executar

Você pode rodar este projeto de duas formas:

### ✔️ Opção 1: Google Colab

1. Acesse o Google Colab
2. Faça upload do arquivo `.ipynb`
3. Execute as células

### ✔️ Opção 2: Ambiente local (VS Code)

* Instale as dependências
* Execute com Python (adaptações podem ser necessárias)
